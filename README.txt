Integrate Rentware into your WordPress website, using the popular Elementor Page Builder

== Description ==

The Rentware Addon for Elementor is a powerful plugin designed to seamlessly integrate Rentware into your WordPress website, using the popular Elementor Page Builder.

**Requirements:**

- You need a **Rentware subscription** to use this plugin. Learn more about Rentware and its pricing at rentware.com.


**Rentware Features:**

- **Elementor Integration**: Easily incorporate Rentware widgets into your Elementor-powered website for a visually pleasing and intuitive user experience.
- **Create Rental Listings**: Design and showcase your rental listings with custom layouts and styles, tailored to match your site's branding.

- **Availability Calendars**: Display real-time availability calendars to assist your customers in making informed booking decisions.

- **User-Friendly**: The addon is designed with simplicity in mind, making it user-friendly for both website administrators and visitors.

== Installation ==

1. Upload the 'rentware-addon-for-elementor' folder to the '/wp-content/plugins/' directory or install it via the WordPress plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.

Getting started with Rentware is easy and takes just a few steps:  

1. **Start a Free Trial:** Begin by signing up for a free trial of Rentware. Start your free trial [here](https://rentware.com/en/free-trial/)  

2. **Set Up Your Rentware Account:** After signing up, set up your Rentware account with the necessary details and configurations.

3. **Grab Your Access Token:** Within your Rentware account, locate and generate your access token.

4. **Place the Rentware Checkout Widget in Elementor:** Using the Elementor Page Builder, simply drag and drop the Rentware checkout widget onto your desired page or post.

5. **Paste Your Access Token:** In the Rentware checkout widget settings, paste the access token you obtained earlier.

You're now ready to start seamlessly integrating Rentware into your website and offering your customers a streamlined rental and booking experience.

If you have any further questions or need assistance, please refer to our documentation or reach out to our support team for guidance.

== Frequently Asked Questions ==

= How do I integrate Rentware widgets into my Elementor pages? =

After activating the plugin, you'll find dedicated Rentware widgets in the Elementor editor. Simply drag and drop them onto your page, configure the settings, and you're ready to go.

= Is this plugin compatible with the latest version of Elementor? =

Yes, the Rentware Addon for Elementor is designed to work seamlessly with the latest versions of both WordPress and Elementor.

== External Services ==

This plugin relies on a 3rd party service, Rentware, for some of its functionality. 

The Rentware service is used to provide real-time availability calendars and a streamlined rental and booking experience. The plugin loads a script from `https://cdn.rtr-io.com/widgets.js` to enable this functionality. Please note that the script is hosted on a different domain than Rentware's website.

For more information about Rentware and its services, please visit [Rentware's website](https://rentware.com/en/).

Before using this plugin, we recommend reviewing Rentware's [Privacy Policy](https://rentware.com/en/privacy-policy/).
